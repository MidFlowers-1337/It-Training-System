# 02. 基础组件与模式 (Primitives & Patterns)

本模块定义了构建应用所需的两类积木：**Primitives (原子组件)** 和 **Patterns (设计模式)**。

## 1. Primitives (原子组件)

基于 PrimeVue Unstyled，仅封装样式，不含复杂业务逻辑。

### 1.1 Button (按钮)

-   **Variants**:
    -   `primary`: 蓝色背景，白色文字 (`bg-brand-blue text-white`)
    -   `secondary`: 灰色背景，黑色文字 (`bg-background-tertiary text-text-primary`)
    -   `ghost`: 透明背景，蓝色文字 (`text-brand-blue hover:bg-background-secondary`)
-   **Size**: 固定高度 `44px` (Touch-friendly)。
-   **Shape**: 圆角 `12px`。

### 1.2 Input (输入框)

-   **Style**: 灰色背景 (`bg-background-secondary`)，无边框。聚焦时显示蓝色光环。
-   **Typography**: `text-[17px]` (防止 iOS 缩放)。
-   **Icon**: 支持前置/后置图标 (如搜索放大镜)。

### 1.3 Divider (分割线)

-   **Style**: `h-[1px] bg-background-tertiary w-full`。
-   **Variant**: `indented` (左侧缩进 16px，用于列表项之间)。

## 2. Patterns (设计模式) - 核心架构

这些是强制使用的布局模式，用于构建所有业务页面。

### 2.1 ListRow (列表行)

这是应用中最核心的模式，用于替代卡片。

```vue
<!-- 伪代码结构 -->
<template>
  <div class="flex items-center py-4 border-b border-background-tertiary hover:bg-background-secondary transition-colors cursor-pointer">
    <!-- 左侧：图标/缩略图 -->
    <div class="w-12 h-12 rounded-lg bg-background-secondary mr-4 flex-shrink-0">
      <slot name="icon" />
    </div>
    
    <!-- 中间：主要信息 -->
    <div class="flex-1 min-w-0">
      <h3 class="text-[17px] font-medium text-text-primary truncate">
        {{ title }}
      </h3>
      <p class="text-[14px] text-text-secondary truncate mt-0.5">
        {{ subtitle }}
      </p>
    </div>
    
    <!-- 右侧：元数据/操作 -->
    <div class="ml-4 flex items-center text-text-secondary">
      <slot name="action" />
      <IconChevronRight class="w-4 h-4 ml-2 opacity-30" />
    </div>
  </div>
</template>
```

### 2.2 PageHeader (页面标题)

统一的页面头部，包含大标题和操作栏。

```vue
<template>
  <div class="mb-8">
    <div class="flex justify-between items-end mb-4">
      <h1 class="text-[40px] font-semibold text-text-primary tracking-tight">
        {{ title }}
      </h1>
      <div class="flex gap-3">
        <slot name="actions" />
      </div>
    </div>
    <Divider />
  </div>
</template>
```

### 2.3 DescriptionList (详情列表)

用于详情页展示信息，替代 Grid 卡片。

```vue
<template>
  <div class="space-y-0">
    <div v-for="item in items" class="flex justify-between py-3 border-b border-background-tertiary last:border-0">
      <span class="text-text-secondary">{{ item.label }}</span>
      <span class="text-text-primary font-medium">{{ item.value }}</span>
    </div>
  </div>
</template>
```

### 2.4 Section (区块容器)

用于分隔页面内容。

-   **Props**: `title` (可选)
-   **Style**: 包含 `mb-16` (64px 间距)。
