# 01. 设计系统 (Design System)

## 1. Design Tokens (TypeScript)

我们将使用 TypeScript 常量来定义 Design Tokens，确保类型安全和统一性。

### 1.1 颜色系统 (Colors & Themes)

我们支持三套主题：**Light (晴空白)**, **Dark (深空黑)**, **Warm (暖阳橙)**。
实现机制：Tailwind 使用 CSS 变量 (e.g., `var(--bg-primary)`)，而这些变量的值由 TypeScript 主题对象动态注入到 `<html>` 标签的 style 属性中。

```typescript
// src/design-system/tokens/colors.ts

export type Theme = 'light' | 'dark' | 'warm';

export const themes: Record<Theme, Record<string, string>> = {
  light: {
    '--bg-primary': '#FFFFFF',
    '--bg-secondary': '#F5F5F7', // Apple Light Gray
    '--bg-tertiary': '#E5E5E5',
    '--text-primary': '#1D1D1F',
    '--text-secondary': '#86868B',
    '--brand-primary': '#0071E3', // Apple Blue
  },
  dark: {
    '--bg-primary': '#000000',   // Pure Black (OLED)
    '--bg-secondary': '#1C1C1E', // Dark Gray
    '--bg-tertiary': '#38383A',
    '--text-primary': '#F5F5F7',
    '--text-secondary': '#86868B',
    '--brand-primary': '#0A84FF', // Lighter Blue for Dark Mode
  },
  warm: {
    '--bg-primary': '#FFF2EB',   // Cream
    '--bg-secondary': '#FFE8CD', // Light Orange
    '--bg-tertiary': '#FFD6BA',
    '--text-primary': '#4A3B32', // Dark Brown
    '--text-secondary': '#8C7B70',
    '--brand-primary': '#FF9500', // Orange
  }
};
```

### 1.2 字体排印 (Typography)

**原则**: 严格的层级。使用系统字体栈。

```typescript
// src/design-system/tokens/typography.ts

export const typography = {
  fontFamily: {
    sans: '-apple-system, BlinkMacSystemFont, "SF Pro Text", "Helvetica Neue", sans-serif',
  },
  // 预定义样式类 (Tailwind classes)
  styles: {
    h1: 'text-[40px] leading-[1.1] font-semibold tracking-tight text-primary', // 页面大标题
    h2: 'text-[28px] leading-[1.2] font-semibold tracking-tight text-primary', // 区块标题
    h3: 'text-[20px] leading-[1.3] font-semibold text-primary',                // 分组标题
    body: 'text-[17px] leading-[1.5] font-normal text-primary',                // 正文
    caption: 'text-[14px] leading-[1.4] font-normal text-secondary',           // 辅助说明
    label: 'text-[12px] leading-[1.3] font-medium uppercase tracking-wider text-secondary', // 标签
  }
}
```

### 1.3 间距与网格 (Spacing & Grid)

**原则**: 4px 基准网格。大留白。

```typescript
// src/design-system/tokens/spacing.ts

export const spacing = {
  layout: {
    pagePadding: '40px',     // 页面左右内边距
    sectionGap: '64px',      // 区块间距
    itemGap: '16px',         // 列表项间距
  },
  component: {
    inputHeight: '44px',     // 输入框/按钮高度
    radius: '12px',          // 通用圆角
  }
}
```

## 2. 动效规范 (Motion)

**原则**: 快速、隐形。仅用于反馈和状态切换。

-   **Duration**: 200ms - 300ms
-   **Easing**: `cubic-bezier(0.25, 0.1, 0.25, 1.0)` (Ease)
-   **Usage**:
    -   Hover: 颜色渐变 (200ms)
    -   Drawer: 从右侧滑入 (300ms)
    -   List: 数据加载时淡入 (Fade In)

## 3. Tailwind 配置策略

我们将编写一个 Tailwind 插件，将上述 Tokens 注入到 Tailwind 的 theme 中，并禁用默认颜色板，强制开发者只使用我们定义的语义化颜色。

```javascript
// tailwind.config.js (伪代码)
module.exports = {
  theme: {
    colors: {
      // 映射到 CSS 变量
      primary: 'var(--bg-primary)',
      secondary: 'var(--bg-secondary)',
      // ...
    },
    extend: {
      // ...
    }
  }
}
```
