## 🎯 项目特色

### 核心亮点

1. **高并发选课系统** ⚡

## 🚀 快速开始

### 环境要求

| 软件 | 版本 |
|------|------|
| JDK | 17+ |
| Node.js | 18+ |
| MySQL | 8.0+ |
| Redis | 7.0+ |
| RocketMQ | 5.1+ |
| Ollama | Latest |

### 5分钟快速启动

```bash
# 1. 克隆参考项目
git clone https://gitee.com/liy6/easy-enroll.git

# 2. 创建数据库
mysql -u root -p
CREATE DATABASE it_training;

# 3. 启动后端
cd backend
mvn spring-boot:run

# 4. 启动前端
cd frontend
npm install
npm run dev

# 5. 访问系统
# 浏览器打开 http://localhost:3000
```

详细步骤请查看 [快速开始指南](./IT技能培训智能选课系统-快速开始.md)

## 💡 核心技术实现

### 高并发选课流程

```
用户请求 
  ↓
令牌桶限流
  ↓
本地缓存防重（第一级）
  ↓
Redis防重（第二级）
  ↓
Redis预减库存
  ↓
发送MQ消息
  ↓
异步消费处理
  ↓
分布式锁
  ↓
数据库防重（第三级）
  ↓
扣减库存 + 创建订单
  ↓
通知用户
```

### AI推荐流程

```
获取学员信息
  ↓
分析技能水平
  ↓
查看学习历史
  ↓
构建AI Prompt
  ↓
调用Ollama模型
  ↓
解析AI响应
  ↓
计算推荐分数
  ↓
返回推荐结果
```
